## Core Concepts

- Findings are raw security signals from AWS services.
- Actions are prioritized remediation units derived from findings.
- Exceptions document risk acceptance with reason and expiry.
- Help cases are requester-private support threads in the current MVP.
- Customer remediation currently runs through reviewed PR bundles; `direct_fix` and `WriteRole` are out of scope.
